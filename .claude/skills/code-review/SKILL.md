# Code Review Skill

Review code for:

## Security

- Hardcoded passwords
- SQL injection
- Secrets in code

## Performance

- Unnecessary loops
- Multiple DB calls
- Large memory usage

## Quality

- Missing type hints
- Missing error handling
- Missing unit tests
